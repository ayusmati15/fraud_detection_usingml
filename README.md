# Credit Card Fraud Detection Using Machine Learning

## 📌 Project Overview

Built a machine learning pipeline to classify fraudulent credit card transactions, benchmarking multiple classification architectures to identify the most robust model for financial risk mitigation.

## 📊 Dataset

* **Source:** Highly imbalanced Kaggle dataset (284,808 transactions, 31 features).


* **Features:** Contains anonymized PCA-transformed variables, `Time`, `Amount`, and a binary target `Class` (1 = Fraud, 0 = Benign).



## ⚙️ Implemented Models

1. **K-Nearest Neighbors (KNN)**

2. **Decision Tree (DT)**

3. **Logistic Regression (LR)**

4. **Support Vector Machine (SVM)**


## 📈 Key Results & Future Scope

* **Performance:** **KNN** and **Decision Tree** models achieved peak accuracy, effectively identifying anomalies within the holdout evaluation set.


* **Next Steps:** Integrate real-time geolocation and telecom telemetry data to automatically flag cross-border transaction anomalies.
